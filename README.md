# Game
Playground for learning TypeScript, Webpack, Sass

## Install
* ```git clone 'path'```
* ```cd game```
* ```npm install``` or ```yarn```

## Scripts
* dev - development mode with live reload (```npm run dev```)
* build - production mode (```npm run build```)

## Webpack aliases
* *"@"* - root/src/
* *"@mixins"* - root/src/script/mixins/
* *"@games"* - root/src/script/games/
